# Projet MOGPL - Optimisation robuste dans l'incertain total

Paul Chambaz & Zélie van der Meer - 2024

# Installation

Ce projet requiert une license Gurobi et la libraire python `pulp`.

```sh
pip install pulp
```

## Partie 1

### Question 1

Commande à exécuter :

```sh
python3 src/q11.py
```

Affichage dans le terminal
